
# ARTEMIS bản hoàn chỉnh server-based

## Có gì trong bộ này
- `public/index.html`
- `public/admin.html`
- `public/logo.png`
- `public/nam.png`
- `server.js`
- `data/signals.json`
- `data/packages.json`
- `.env.example`

## Chạy local
```bash
npm install
cp .env.example .env
npm run dev
```

## Truy cập
- Web: `http://localhost:3000`
- Admin: `http://localhost:3000/admin.html`
